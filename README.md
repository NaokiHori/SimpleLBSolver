## Overview

A toy Rayleigh-Benard simulator by means of the Lattice-Boltzmann method.

## Requirements

- C compiler
- GNU make

## Quick Start

### Fetch Source

```console
git clone https://github.com/NaokiHori/SimpleLBSolver
cd SimpleLBSolver
```

### Build and Run

```console
make output
make all
./a.out
```

Edit `Makefile` to enable OpenMP parallelization.
